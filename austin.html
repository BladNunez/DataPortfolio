Project Overview

In this project I analyze bikeshare trips in the city of Austin,TX from 2013 - 2024. In the year 2024 our data is only updated up until February. In this project the goals are listed down below:
1.) Analyze bike trip durations to understand average trip lengths and peak usage times.
2.) Identify popular start and end stations to optimize bike distribution and station maintenance.
3.) Explore subscriber types to understand user demographics and preferences.
4.) Visualize bike usage trends over time to identify seasonal variations.

Challenges

One challenge I ran into was cleaning out the data that was in my station name columns. In the start column there was stations that were the same name but either in diffrent order or in one word in some cases. I manually went into the data and fixed each one by one so my data was consistent and my analysis accurate.
